# ✂️ PWA: Peluquería Andrea Carvajal

Esta Aplicación Web Progresiva (PWA) es un sistema inteligente de gestión de citas, servicios y stock para salón de belleza. 

## 🎓 Créditos y Autoría Institucional

Este proyecto ha sido desarrollado en el marco del programa de innovación educativa **"CITE-STEAM IA, FP y robótica inclusiva"** del IES San José (Badajoz).

* **Autora y Creadora de la App:** Andrea Carvajal González
* **Nivel Educativo:** 2º Curso del Ciclo Formativo de Grado Medio de Peluquería y Cosmética Capilar.
* **Coordinación del Módulo (Digitalización):** Sofía Merino Tena
* **Coordinación General del Proyecto:** Cristina Ramos García

---
*Desarrollado integrando Inteligencia Artificial (Prompt Engineering), Google Apps Script y arquitecturas web móviles.*
